# Кинотеатр «Места для поцелуев» 

REST API для системы бронирования и продажи билетов в кинотеатре.

## Основные возможности

- Управление залами, рядами и местами
- Работа с фильмами и сеансами
- Бронирование и продажа билетов
- Контроль пересечений сеансов
- Аудит изменений
- Резервное копирование в JSON
- Интеграция с Telegram-ботом (в будущем)

##  Технологии

- **Язык**: Java 17
- **Фреймворк**: Spring Boot
- **База данных**: PostgreSQL (через Docker)
- **Сборка**: Maven
- **Контейнеризация**: Docker Compose
- **Инструменты**: IntelliJ IDEA, Postman

## Запуск проекта

1. Запусти PostgreSQL в Docker:
  
   docker compose up -d
